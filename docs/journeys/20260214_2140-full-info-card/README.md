# 旅程: 全能力画像 — 完整交互信息卡片

## 画像

**全能力 {states, curves, patch, folme}** — 四种能力全覆盖

## 设计目标

信息展示卡片: 默认简洁态 → 状态 2 展开详情态。
弹簧曲线控制展开弹性 (响应 0.48, 阻尼 0.88)。
Touch → To 实现点击展开。
Folme 验证: 弹簧引擎参数面板完整性。

白色容器 (圆角 12px) + 紫色头部色块 (嵌套)。
