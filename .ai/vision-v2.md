# Intera V2 Visual Redesign

## WHY
当前 UI 是"功能原型"级别，视觉品味不够现代。kenefe 明确要求提升审美。

## 核心洞察
**状态 = 关键帧 = 那一行画布**。状态条不是 tab 切换，是关键帧缩略图条。

## 问题清单
1. 色彩层次差太小，灰蒙蒙
2. 工具栏太矮太挤 (40px)
3. 字号太小 (11-12px)
4. 面板之间没有呼吸感
5. 状态条太简陋，应该是关键帧缩略图
6. Patch 工具栏文字按钮像 debug 面板

## 实施顺序
Phase 1: Design Token 升级（色彩/字号/间距/圆角）
Phase 2: 状态条 → 关键帧缩略图条
Phase 3: 工具栏图标化 + 面板层次优化
